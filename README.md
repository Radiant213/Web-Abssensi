# SecureGate IoT Dashboard (Frontend & Backend)

Ini adalah folder core untuk SecureGate IoT Attendance System. Menggunakan Next.js 15 dengan Custom Server Node.js/Express.

## Cara Menjalankan
1. Pastikan database MySQL sudah aktif.
2. Edit `.env.local` jika diperlukan.
3. Jalankan perintah:
```bash
npm install
npm run build
npm start
```

Lihat [README.md utama](../README.md) di folder root untuk informasi selengkapnya.
